# COCORO Official LP

Girls Hip Hop Dancer COCORO のオフィシャルランディングページ。

## 構成

```
cocoro-lp/
├── index.html        # メインページ
├── assets/
│   └── profile.png   # プロフィール写真（背景透過済み）
├── .gitignore
└── README.md
```

## 技術スタック

- HTML / CSS / Vanilla JS
- Google Fonts（Bebas Neue / Dancing Script / Oswald / Noto Sans JP）
- Instagram埋め込み（Reels）
- Canvas API（グリッターパーティクル）

## デプロイ

XServer にてホスティング。  
ドメイン: cocoro-dance.com（予定）

## 素材差し替え箇所

- `assets/profile.png` → プロフィール写真
- EVENTセクションの日程・会場 → 都度更新
- CONTACTのメールアドレス → 実際のアドレスに変更
